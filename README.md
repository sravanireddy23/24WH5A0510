# Campus Notifications Frontend

Frontend implementation for the **Affordmed Campus Evaluation** task — a campus notifications dashboard built with **React + Vite + Material UI**.

## GitHub Repository
Repository created for submission: **24WH5A0510**

---

## Features Implemented

- Notifications page UI
- Clean Material UI based layout
- Notification filter options
- Notification card structure
- Pagination UI
- Loading / error / empty state handling
- API integration structure for:
  - registration
  - authentication
  - notifications fetch

---

## Tech Stack

- **React**
- **Vite**
- **Material UI (MUI)**
- **Axios**
